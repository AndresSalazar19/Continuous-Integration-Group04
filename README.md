# CaféLibro — Library Loan Manager

| Feature | Description                     | Owner                    |

| 1       | Register a member               | Andrés Salazar           |

## Setup

```bash
python -m venv venv
.\venv\Scripts\activate.bat
pip install -r requirements.txt
pytest -q
```
